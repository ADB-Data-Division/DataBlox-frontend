datablox_od.display#

Module contents#

Provides functions for creating data visualizations.

datablox_od.display.plot_on_map(df, column=None, width=8, height=8)#

Plots a GeoPandas data frame on a map. Categories can be defined via the column parameter.

Parameters:

  • df (geopandas.geodataframe.GeoDataFrame) – Data frame to be plotted.

  • column (str | None) – Name of the column containing the categories. If the value passed is not None, then the colors in the plot correspond to the unique values in this column.

  • width (int) – Width of the plot.

  • height (int) – Height of the plot.
